# 枕下压电 BCG 呼吸信号提取研究方案

## 0. 论文研究背景与主线

本研究面向睡眠场景下的枕下压电 BCG 信号，关注其在非接触或弱接触条件下承载呼吸努力信息的机制与建模方法。枕下 BCG 信号并不是单一呼吸源的直接观测，而是呼吸运动、心冲击、头颈接触状态、机械传递路径和噪声共同作用后的混叠信号。传统低通滤波、EWT/IEWT 等方法依赖固定频带或频谱边界，在呼吸幅值较低、频率漂移、心冲击调制明显或信号形态非平稳时，容易出现频谱错分、双峰伪影和幅值趋势失真。

因此，本文的学术问题不是构建一个覆盖所有异常情况的工业级鲁棒系统，而是研究：

> 在 PSG 胸带、腹带弱同步参考条件下，如何从单通道枕下压电 BCG 中学习连续的呼吸努力估计信号，并解释胸带、腹带信息如何参与综合呼吸努力信号的构造。

研究主线应保持连贯：

```text
BCG 混叠观测
  -> 弱同步 PSG 胸带/腹带监督
  -> 胸带相关与腹带相关中间估计
  -> 综合呼吸努力构造
  -> 呼吸节律、努力包络和频谱结构评价
```

体动剔除、信号质量、心冲击可用性和低可信片段等问题需要被诚实说明，但它们不构成本文的研究主体。它们更适合作为实验对比、敏感性分析或讨论章节中的边界条件，用于解释模型在哪些条件下表现更好或更差，而不是把研究扩展成“BCG 可靠时如何做、不可靠时如何兜底”的工程化系统设计。

## 1. 研究定位

本课题面向枕下压电传感采集的 BCG 混叠信号，目标是从其中提取呼吸努力估计信号，用于反映受试者的呼吸节律和呼吸努力变化。

在当前数据条件下，任务不宜定义为“从 BCG 精确还原 PSG 胸带/腹带原始波形”，而应定义为：

> 基于弱同步 PSG 参考的枕下压电 BCG 呼吸努力估计信号建模。

模型需要输出的是一个在 BCG 时间轴上的连续呼吸努力估计信号，并可同时估计呼吸率和努力包络等辅助结果。该输出应在窗口级、趋势级和频谱级与 PSG 胸带/腹带参考一致，而不是追求逐点相位完全一致。

## 2. 数据前提与核心约束

### 2.1 PSG 与 BCG 未精确同步

两路信号未能获得精确同步，因此不应直接使用逐点波形监督作为主要训练目标。

不推荐的目标：

```text
BCG(t) -> 胸带(t)
BCG(t) -> 腹带(t)
BCG(t) -> RIPsum(t)
```

不推荐的主损失：

```text
MSE(预测呼吸波形, PSG 胸/腹带波形)
```

即使模型预测出了正确的呼吸节律，只要 BCG 与 PSG 存在秒级错位，逐点 MSE 也会错误惩罚模型，导致模型学习出过度平滑、幅值被压缩的平均波形。

### 2.2 体动已在信号处理初期剔除

体动剔除是本文的数据预处理前提，不作为本文的核心研究对象。需要保留以下信息，以避免弱同步监督和整夜评价被破坏：

- 原始时间戳；
- clean mask；
- 被剔除片段的起止位置；
- clean segment 与原始记录的映射关系；
- clean segment 边界附近是否存在体动剔除。

体动段的处理原则是保持原始时间关系清晰，而不是围绕其设计复杂兜底流程。若体动段已经被删除，应维护片段映射表；若体动段保留，则在训练和评价中使用 clean mask。体动剔除比例、边界影响和片段可用性可作为对比分析或讨论内容，不作为主模型设计目标。

### 2.3 单通道枕下信号的可辨识性有限

单通道枕下 BCG 不一定能唯一还原胸带和腹带两路独立运动。更稳妥的主目标是估计综合呼吸努力对应的呼吸努力估计信号。

但这不意味着不能利用胸带和腹带。更合适的路线是：先以 PSG 胸带和腹带分别构造弱同步监督信号，训练模型学习胸部呼吸估计分支和腹部呼吸估计分支；再比较不同综合构造策略，形成最终的呼吸努力估计信号。

因此，胸带/腹带估计应被定位为中间输出和可解释性分析对象，而不是当前研究必须证明其精确还原的最终目标。最终评价重点仍放在综合呼吸努力估计信号的节律、包络和频谱结构上。

## 3. 研究目标分层

### Level 1：呼吸节律估计

目标：

- 呼吸率轨迹；
- 呼吸周期长度；
- 呼吸主频变化；
- 窗口级呼吸次数。

该层最稳定，适合作为基础目标。

### Level 2：呼吸努力幅值趋势估计

目标：

- 呼吸努力强弱变化；
- 呼吸努力幅值减弱；
- 呼吸努力恢复性增强；
- 周期性呼吸或努力波动；
- 局部努力包络变化。

该层是课题的核心，但应采用相位不敏感监督，重点比较趋势、相对幅值和局部变化。

### Level 3：综合呼吸努力构造

目标：

- 分别估计与胸带、腹带相关的呼吸努力信号；
- 比较胸带估计、腹带估计与综合信号之间的关系；
- 研究归一化相加、质量加权、包络融合和学习式融合等综合构造路径；
- 得到最终用于评价的综合呼吸努力估计信号。

该层是方法研究的一部分，但不应被表述为“单通道 BCG 必然可以精确分离胸腹两路”。胸带/腹带估计结果首先用于辅助建模和融合分析，最终结论应落在综合呼吸努力估计信号上。

## 4. 信号信息利用原则

### 4.1 BCG 是耦合观测，不是简单加性噪声

枕下压电 BCG 可近似理解为以下成分的混合：

```text
呼吸运动 + 心冲击图 + 体动残余 + 接触状态变化 + 机械传递噪声 + 环境噪声
```

这些成分不是完全独立的。呼吸会调制心冲击幅值、基线、形态和能量分布；体位、枕头接触、头颈位置也会改变传递路径。因此，前处理阶段不宜强行做硬分离。

推荐做法是模型内部软分解：

```text
raw BCG
  -> respiratory latent branch
  -> thoracic-related branch
  -> abdominal-related branch
  -> optional high-frequency context branch
  -> time-frequency branch
  -> optional context branch
  -> gated fusion
  -> respiratory effort estimation signal / RR / envelope
```

### 4.2 呼吸分量

呼吸分量是主输出，但应定义为 BCG 时间轴上的呼吸努力估计信号，而不是 PSG 胸带或腹带的复制品。

建议输出：

- `r_hat(t)`：呼吸努力估计信号；
- `A_hat(t)`：呼吸努力包络；
- `RR_hat(t)`：呼吸率轨迹；
- `P_breath(t)`：呼吸峰/谷或周期概率；

可增加探索性中间输出：

- `r_tho_hat(t)`：胸带相关呼吸估计信号；
- `r_abd_hat(t)`：腹带相关呼吸估计信号。

这两个中间输出用于利用 PSG 胸带和腹带的监督信息，并为综合信号构造提供候选输入。它们不应被解释为模型已经完成了胸带/腹带的确定性分离。

### 4.3 心冲击分量

心冲击不应简单视为噪声，但在当前数据集中也不应被设为必要输入或必要模块。若心冲击成分并不总是清晰可用，模型不应强依赖 J 峰、心搏位置或手工心冲击包络。

高频或心冲击候选频段的 CWT 时频图可以作为优先选择。CWT 幅值图已经包含：

- 高频能量随时间的变化；
- 各频率分量的幅度分布；
- 高频能量是否存在周期性；
- 高频能量是否被呼吸节律调制；
- 高频噪声、接触变化或残余运动的宽带能量特征。

心冲击包络本质上可以看作对心冲击候选频段时频能量的频率方向聚合：

```text
cardiac_envelope(t) = aggregate(CWT(t, f)), f in cardiac band
```

因此，当模型已经输入高频 CWT 幅值图时，额外输入心冲击包络可能是冗余的，也可能在心冲击不可用片段引入错误先验。

推荐定位：

- 必选：原始 BCG、呼吸候选频带和时频图；
- 可选：数据状态或上下文特征；
- 可选：高频/心冲击候选频段 CWT；
- 消融项：心冲击频带包络、J 峰、HR、心搏位置；
- 仅在 ECG 或心搏参考可靠时，才考虑心冲击辅助监督。

### 4.4 时频信息

时频图有价值，但不应作为唯一输入，也不应以“修复时频图再反变换”为主路线。

更合理的使用方式：

- 辅助识别呼吸频谱 ridge；
- 辅助观察呼吸频带能量连续性；
- 辅助区分呼吸、心冲击和残余噪声；
- 用窗口级频谱结构监督模型。

时频特征应与时域原始信号和呼吸候选频带共同输入模型。高频 CWT、数据状态特征可作为辅助上下文；心冲击包络不作为默认必选输入。

### 4.5 接触状态与低频漂移

低频漂移不一定全部是噪声，也可能反映：

- 头颈位置变化；
- 枕头受力变化；
- 体位变化；
- 传感器接触状态变化；
- 机械传递路径变化。

这类信息不适合作为呼吸主输出，但适合作为上下文特征，用于解释片段间差异。

## 5. PSG 参考标签构造

由于同步不精确，应优先从 PSG 胸带、腹带构造相位不敏感标签。

### 5.1 综合努力参考

综合呼吸努力参考不应简单固定为胸带和腹带的加性结果。胸带与腹带可能存在幅值差异、质量差异和局部相位差，直接相加有时会造成抵消或错误增强。

可比较以下构造方式：

```text
1. 归一化相加:
   RIPsum = normalized(胸带) + normalized(腹带)

2. 质量加权融合:
   R_effort = w_tho * normalized(胸带) + w_abd * normalized(腹带)

3. 包络融合:
   A_effort = combine(A_tho, A_abd)

4. 呼吸率/频谱层面鲁棒融合:
   RR_ref / Ridge_ref = robust_combine(胸带特征, 腹带特征)
```

其中归一化相加可作为基线，但不应被默认视为唯一正确的综合努力参考。当前更推荐在特征层面保留胸带、腹带各自的信息，再比较不同融合策略对最终呼吸努力估计信号的影响。

注意不要只做全局归一化后丢弃幅值变化。呼吸努力减弱、恢复性增强等目标需要保留相对幅值趋势。

### 5.2 推荐标签

从 PSG 中提取：

- `RR_tho[k] / RR_abd[k]`：胸带、腹带各自的窗口级呼吸率；
- `RR_ref[k]`：窗口级呼吸率；
- `A_tho[k] / A_abd[k]`：胸带、腹带各自的努力幅值；
- `A_ref[k]`：窗口级努力幅值；
- `Drop_ref[k]`：相对幅值下降比例；
- `Spec_ref[k, f]`：呼吸频带归一化谱；
- `Ridge_ref[k]`：主呼吸频率；
- `Recovery_ref[k]`：恢复呼吸增强标签。

其中 `RR_ref`、`A_ref`、`Spec_ref` 和 `Ridge_ref` 可以来自胸带/腹带的质量加权融合、鲁棒平均或学习式融合。胸带、腹带各自的标签用于训练中间分支和分析融合路径；综合标签用于训练最终呼吸努力估计信号。

这些标签比原始逐点波形更适合弱同步数据。

### 5.3 同步可信度分级

建议为每一晚或每个 clean segment 建立同步可信度等级。

#### S2：局部同步可信

特征：

- BCG 低频呼吸候选信号与 PSG 胸带、腹带或综合参考存在稳定高相关；
- 最佳 lag 较稳定；
- 局部峰谷关系基本可信。

可用监督：

- shift-invariant waveform loss；
- 峰/谷弱监督；
- 包络损失；
- 呼吸率损失；
- 频谱损失。

#### S1：粗同步可信

特征：

- 知道 BCG 与 PSG 属于大致相同时间段；
- 秒级峰谷不可靠；
- 窗口级统计仍可信。

可用监督：

- 呼吸率损失；
- 包络趋势损失；
- 频谱 ridge 损失。

不建议使用逐点波形损失。

#### S0：同步关系不可信

特征：

- 无法确认 BCG 与 PSG 的时间对应关系；
- 局部或窗口级对应都不可靠。

可用方式：

- BCG 自监督预训练；
- record-level 标签；
- 表征学习；
- 整夜呼吸节律统计标签。

不应声称从 S0 片段学习到了逐时呼吸努力变化。

## 6. 推荐模型：WeakSync-RespBCG-Net

### 6.1 输入

```text
x_raw       : 原始 BCG clean 片段
x_resp      : BCG 呼吸候选频带
x_hf_cwt    : 高频/心冲击候选频段 CWT 时频图（可选）
TF          : 全频或呼吸相关 STFT / CWT / SST 时频图
context_pre : rolling energy / STD / derivative energy / clean mask（可选，用于补充分析）
```

其中 `context_pre` 主要用于描述数据状态和进行对比分析，不应把模型主线扩展为复杂的质量兜底系统。

### 6.2 编码器

- 1D Temporal Encoder：学习时域节律、局部形态和长程变化；
- 2D Time-Frequency Encoder：学习呼吸 ridge、频带能量和非平稳变化；
- High-Frequency Context Encoder：可选，用于学习高频/心冲击候选频段的时频上下文；
- Quality/Context Encoder：可选，用于分析接触状态、低频漂移和片段差异对结果的影响。

### 6.3 融合方式

推荐使用 gated fusion 或 cross-attention。融合模块的核心作用是整合时域、时频域以及胸带/腹带相关中间估计，而不是构建复杂的质量兜底策略。

例如：

```text
raw/time branch + TF branch + optional high-frequency TF branch + optional context branch
                    -> gated fusion
                    -> multi-task heads
```

若加入胸带/腹带中间估计，可采用两级融合：

```text
shared respiratory representation
  -> thoracic head: r_tho_hat(t)
  -> abdominal head: r_abd_hat(t)
  -> fusion module: r_hat(t), A_hat(t), RR_hat(t)
```

融合模块可比较多种构造策略，包括归一化相加、质量加权、包络融合和学习式融合。该比较本身可以作为方法研究的一部分。

### 6.4 输出

```text
r_hat(t)     : BCG 时间轴上的呼吸努力估计信号
A_hat(t)     : 呼吸努力包络
RR_hat(t)    : 呼吸率轨迹
P_breath(t)  : 呼吸峰/谷或周期概率
```

探索性中间输出：

```text
r_tho_hat(t) : 胸带相关呼吸估计信号
r_abd_hat(t) : 腹带相关呼吸估计信号
```

最终输出仍以 `r_hat(t)` 为主。`r_tho_hat(t)` 和 `r_abd_hat(t)` 用于辅助监督、融合分析和可解释性讨论，不作为当前研究必须稳定还原的最终目标。

可选输出：

```text
Q_hat(t)          : 质量或不确定性指标（用于对比分析）
HF_context(t)     : 高频上下文指标（用于对比分析）
Card_env_hat(t)   : 心冲击包络（仅作为消融项）
HR_hat(t)         : 心率轨迹（仅在 ECG 或心搏参考可靠时）
J_peak_hat(t)     : 心搏位置概率（仅在 ECG 或心搏参考可靠时）
```

## 7. 窗口设计

推荐采用多尺度窗口。

### 短窗：8-16 秒

用于：

- 局部波形；
- 高频局部结构；
- 呼吸峰谷；
- 短时片段特征分析。

### 中窗：30-60 秒

用于：

- 呼吸率估计；
- 呼吸频谱 ridge；
- 包络趋势。

### 长窗：90-180 秒

用于：

- 呼吸努力持续减弱趋势；
- 呼吸努力恢复性增强；
- 周期性呼吸；
- 长程努力变化。

训练时不建议只随机切短窗，否则容易丢失长程趋势和幅值变化。

## 8. 损失函数设计

总损失可设计为：

```text
L =
  λ1 * L_RR
+ λ2 * L_env
+ λ3 * L_spec
+ λ4 * L_shift_wave
+ λ5 * L_smooth
+ λ6 * L_tho_abd_aux
```

其中 `L_shift_wave` 只用于同步可信片段，不作为全数据主损失。心冲击相关辅助损失和不确定性损失不作为默认项，只作为对比实验或扩展分析加入。

### 8.1 呼吸率损失

```text
L_RR = MAE(RR_hat, RR_ref)
```

若存在局部错位，可在受限窗口内使用允许偏移的损失：

```text
L_RR = min over τ MAE(RR_hat(t), RR_ref(t + τ))
```

也可在 RR 轨迹上使用 Soft-DTW，但需要限制 warping window，避免过度扭曲。

### 8.2 包络趋势损失

```text
L_env = 1 - Corr(A_hat, A_ref)
```

或使用相对幅值下降：

```text
Drop = 当前努力幅值 / 局部基线努力幅值
L_drop = Huber(Drop_hat, Drop_ref)
```

该损失适合努力下降和恢复呼吸增强任务。

### 8.3 频谱结构损失

可比较呼吸频带内的归一化频谱：

```text
L_spec = KL(P_hat(f | window), P_ref(f | window))
```

也可比较主呼吸频率：

```text
L_ridge = MAE(f_resp_hat, f_resp_ref)
```

频谱损失只作为结构约束，不能替代包络和呼吸率监督。

### 8.4 Shift-invariant waveform loss

仅用于 S2 同步可信片段：

```text
L_shift = min over τ [1 - Corr(r_hat(t), y_ref(t + τ))]
```

可在最佳 lag 下附加 Huber 或 L1，但不建议在全部数据上使用。

### 8.5 不确定性分析项

若需要讨论信号质量对结果的影响，可额外输出 `Q_hat(t)` 或不确定性 `s(t)`，并作为对比分析项。可采用：

```text
L = exp(-s) * L_task + s
```

该形式可以分析低质量片段对估计结果的影响，但不应成为本文主模型的必要组成部分。

### 8.6 高频/心冲击辅助项

高频/心冲击信息优先通过 CWT 时频图输入，不默认要求额外构造心冲击包络、J 峰或 HR 标签。

建议做消融比较：

```text
A: raw BCG + respiratory band
B: A + full / high-frequency CWT
C: B + cardiac-band pooled envelope
D: B + ECG/心搏辅助监督（仅在参考可靠时）
```

若有 ECG 或可靠心搏参考，可作为扩展项加入：

- HR 损失；
- J 峰检测损失；
- 心搏间期损失。

若没有可靠心搏参考，不建议强行引入心冲击辅助监督。心冲击包络最多作为可选派生特征或消融实验项；只有当实验表明它能稳定提升呼吸率、努力包络或数据状态分层表现时，再保留。

### 8.7 胸带/腹带辅助损失

若加入 `r_tho_hat(t)` 和 `r_abd_hat(t)`，可分别使用 PSG 胸带和腹带构造弱同步辅助损失：

```text
L_tho_abd_aux =
  L_weak(r_tho_hat, thoracic_ref)
+ L_weak(r_abd_hat, abdominal_ref)
```

这里的 `L_weak` 不应默认采用逐点 MSE，而应优先使用呼吸率、包络、频谱结构和 shift-invariant waveform loss。该损失的作用是约束中间分支学习胸带/腹带相关信息，不代表最终任务变成精确还原两路 PSG 波形。

综合输出 `r_hat(t)` 可由 `r_tho_hat(t)`、`r_abd_hat(t)`、上下文特征和共享呼吸表示共同构造，并通过综合参考标签进行监督。

## 9. GAN 与对比学习的定位

### 9.1 GAN 不宜作为主路线

GAN 容易生成形态上合理但生理上不可靠的呼吸波形。这种“漂亮但错误”的输出可能掩盖真实的努力减弱、恢复性增强或幅值波动。

若使用 GAN，应满足：

- 只作为小权重正则项；
- 判别器可 conditional on BCG 和上下文特征；
- 主监督仍为 RR、包络和频谱 ridge；
- 必须用信号估计指标验证，而不能只看波形是否平滑。

### 9.2 对比学习适合预训练

对比学习适合在同步不精确、标注有限的条件下学习稳定表征。

可设计正样本：

- 同一 BCG 片段的不同增强版本；
- 同一粗时间窗口内的 BCG 与 PSG 特征；
- 同一受试者同一睡眠阶段的相邻片段；
- 呼吸率或努力包络相近的片段。

可设计增强：

- 幅值缩放；
- 时间裁剪；
- 轻微时间扭曲；
- 噪声注入；
- 频带遮蔽；
- 时频图遮蔽。

对比学习用于 encoder 预训练，最终仍需要 PSG 弱同步标签进行监督微调。

## 10. 当前研究边界

当前研究只关注呼吸努力估计信号提取，不把其他下游任务作为模型输出、训练目标或主评价内容。

因此，当前模型只围绕信号估计本身展开，重点输出：

- 呼吸努力估计信号；
- 呼吸率；
- 呼吸努力包络；
- 呼吸峰/谷或周期概率。

可选地，模型可以输出胸带相关和腹带相关的中间估计信号，用于辅助监督和综合信号构造研究；但最终研究对象仍是综合呼吸努力估计信号。

其他下游应用可作为后续独立课题或应用扩展，但不纳入本文档定义的当前研究任务。当前任务的核心是证明：BCG 中可以提取出稳定、连续、与 PSG 胸/腹带弱同步参考一致的呼吸努力估计信号。

## 11. 评价指标

### 11.1 呼吸节律指标

- RR MAE；
- RR RMSE；
- breath count error；
- dominant respiratory frequency error。

建议按 30 秒或 60 秒窗口评价。

### 11.2 努力幅值指标

- 包络趋势相关性；
- 幅值下降检出能力；
- 恢复性增强检出能力；
- 局部相对幅值误差；
- 周期性呼吸趋势一致性。

### 11.3 频谱结构指标

- 呼吸主频误差；
- 呼吸频带能量分布一致性；
- 时频 ridge 连续性；
- 多分辨率频谱相似性。

### 11.4 数据覆盖与敏感性分析

由于体动已剔除，可在补充实验或讨论中报告：

- 有效分析时长占比；
- 被剔除片段占比；
- 每夜可用数据比例；
- 不同数据状态分层下的性能。

这些指标用于解释实验样本边界和结果适用范围，不作为本文主模型的核心评价目标。

### 11.5 波形指标

波形相关性只能作为辅助指标，并且需要满足：

- 只在 S2 同步可信片段计算；
- 允许最佳 lag 对齐；
- 不作为主要结论依据。

## 12. 实施路线

### Step 1：同步可信度审计

对每晚和每个 clean segment：

- 提取 BCG 低频候选呼吸信号；
- 构造 PSG 胸带、腹带及候选综合参考；
- 计算 cross-correlation；
- 估计最佳 lag；
- 统计最大相关系数；
- 观察 lag 是否稳定。

输出：

- 每晚 lag 分布；
- 每段 lag 分布；
- S2/S1/S0 片段比例；
- 是否存在固定 offset、缓慢 drift 或局部同步失效。

### Step 2：构造弱同步标签

从 PSG 胸带/腹带提取：

- `RR_tho / RR_abd`；
- `A_tho / A_abd`；
- `RR_ref`；
- `A_ref`；
- `Drop_ref`；
- `Spec_ref`；
- `Ridge_ref`；
- `Recovery_ref`。

其中胸带/腹带各自标签用于辅助分支，综合标签用于最终呼吸努力估计信号。不要先训练逐点波形回归。

### Step 3：建立传统基线

实现并统一评价：

- 低通滤波；
- EWT；
- IEWT；
- CWT/SST ridge tracking；
- Hilbert-based RR；
- 简单包络估计。

传统方法也应使用弱同步评价指标，避免用未对齐波形相关性直接否定其表现。

### Step 4：训练弱同步多任务模型

第一阶段模型输出：

- RR；
- effort envelope；
- spectral ridge；
- optional context indicators。

此阶段不强求 `r_hat(t)` 与 PSG 逐点对齐。

同时应进行输入消融，判断高频信息是否真的带来增益：

- `raw BCG + respiratory band`；
- 上述输入 + 全频或高频 CWT；
- 上述输入 + 心冲击频带聚合包络；
- 上述输入 + ECG/心搏辅助监督，仅在参考可靠时使用。

若心冲击包络或心搏辅助监督没有稳定提升呼吸率、努力包络、频谱结构或数据状态分层表现，则不应纳入默认模型。

### Step 5：加入胸带/腹带辅助分支

在共享呼吸表示稳定后，加入：

```text
r_tho_hat(t)
r_abd_hat(t)
```

这两个分支分别由胸带、腹带弱同步标签监督，用于学习胸部和腹部呼吸运动相关信息。它们是中间输出，不作为最终结论的主要评价对象。

### Step 6：构造综合呼吸努力估计信号

当 RR、包络和频谱结构预测稳定后，再加入：

```text
r_hat(t)
```

该输出可由共享呼吸表示、胸带辅助分支、腹带辅助分支和上下文特征共同构造。需要比较以下构造策略：

- 归一化相加；
- 质量加权融合；
- 包络融合；
- 学习式融合模块。

该输出使用以下约束：

- shift-invariant waveform loss；
- smoothness prior；
- spectral consistency；
- envelope consistency；
- breath peak weak labels。

不要使用全数据逐点 MSE 作为主监督。

### Step 7：整夜连续评估

在完成片段级建模后，应评估模型在整夜数据上的连续输出质量：

- `r_hat(t)`；
- `A_hat(t)`；
- `RR_hat(t)`；
- context / clean mask。

最终评价呼吸节律、努力包络、频谱结构和波形一致性。有效覆盖率和数据状态分层可作为补充分析。

## 13. 关键假设与对比讨论

### 13.1 未精确同步导致错误监督

讨论点：

- 逐点 MSE 将同步误差误判为模型误差；
- 模型输出被迫平滑化；
- 呼吸峰谷和幅值下降被削弱。

论文处理：

- 建立同步可信度分级；
- 主用窗口级、频谱级和包络级监督；
- 波形损失只用于 S2 片段。

### 13.2 单通道 BCG 无法唯一还原胸腹两路

讨论点：

- 模型根据人群先验猜测胸/腹分量；
- 胸腹反常运动无法可靠恢复。

论文处理：

- 第一目标设为综合呼吸努力估计信号；
- 胸带/腹带估计作为中间输出和融合分析对象；
- 不确定性分析作为补充讨论，而不是主模型目标。

### 13.3 时频图修复可能丢失相位和幅值意义

讨论点：

- 频谱看似合理，但时域幅值和波形变化不可信；
- 反变换后的时域波形缺乏生理解释。

论文处理：

- 时频图作为辅助输入；
- 输出仍以 RR、包络和呼吸努力估计信号为主；
- 频谱损失不单独作为主目标。

### 13.4 GAN 作为生成约束的局限

讨论点：

- 生成正常呼吸样式；
- 掩盖努力减弱、恢复性增强或局部幅值波动；
- 使呼吸努力估计结果偏乐观。

论文处理：

- GAN 仅作为弱正则；
- 主监督保持在呼吸率、努力包络和频谱结构目标；
- 用信号估计指标验证。

### 13.5 体动剔除导致评价偏差

讨论点：

- 只在容易片段上评价；
- 大量剔除导致整夜可用性被高估。

论文处理：

- 在补充实验中报告有效覆盖率；
- 在讨论中分析数据状态分层对结果的影响。

## 14. 最终建议

本课题的核心主张应从“恢复一个漂亮的呼吸波形”转为：

> 模型能够从枕下压电 BCG 中提取呼吸努力估计信号，稳定估计呼吸节律、努力幅值变化和频谱结构；在体动剔除后的有效片段中，这些输出在窗口级、趋势级和频谱级与 PSG 胸/腹带参考一致，并优于低通、EWT、IEWT 等传统方法。

推荐主路线：

```text
BCG 多视图输入
  -> 弱同步多任务网络
  -> 胸带/腹带相关中间估计
  -> 综合构造模块
  -> 呼吸努力估计信号 / RR / 努力包络
  -> 整夜连续呼吸努力估计
```

推荐模型重点：

- 不以逐点波形还原为核心；
- 不把时频图修复作为唯一目标；
- 不把心冲击包络、J 峰或 HR 设为必要依赖；
- 不把胸带/腹带中间估计解释为确定性分离结果；
- 不把体动或低质量片段的兜底处理作为研究主体；
- 优先学习相位不敏感的呼吸节律、幅值趋势和频谱结构。

这一路线更符合当前数据条件，也更容易形成清晰、可信、可评价的研究结果。
